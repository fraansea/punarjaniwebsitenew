### Local Development

Simply open the `index.html` file in your browser:

```bash
# Using Python (Python 3)
python -m http.server 8000

# Using Node.js (with http-server)
npx http-server

# Using PHP
php -S localhost:8000
```

Then navigate to `http://localhost:8000`
